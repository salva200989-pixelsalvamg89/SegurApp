## Generar APK

El APK se genera automáticamente con GitHub Actions al subir el código.

1. Ve a la pestaña **Actions** en GitHub
2. Selecciona el workflow **Build SegurApp APK**
3. Pulsa **Run workflow**
4. Espera ~5 minutos
5. Descarga el APK desde **Artifacts**
